VGCSpreader is an algorithm to create efficient spreads for Pokemon in the VGC format. It currently is in a testing phase and has limited capabilities. As of now only some items and abilities are supported:

**ABILITIES**  
Blaze  
Fairy Aura  
Shadow Shield  
Dark Aura  
Overgrow  
Adaptability  
Prism Armor  
Wonder Guard  
Cloud Nine  
Air Lock  
Multiscale  
Filter  
Solid Rock  
Turboblaze  
Teravolt  
Levitate  

**ITEM**  
Life Orb  
Assault Vest  
Choice Band  
Choice Specs  
All the damage reduction berries (es: Payapa Berry)  

All the damaging moves are supported, moves with secondary effects are generally not supported with the exception of:

**MOVES**  
Acrobatics  
Darkest Lariat  
Facade  
Brine  
Water Spout  
Sacred Sword  
Psyshock  
Foul Play  
Eruption  
Knock Off  
Moongeist Beam  
Menacing Moonraze Maelstrom  
Sunsteel Strike  
Searing Sunraze Smash
